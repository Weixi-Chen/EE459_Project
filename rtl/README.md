# `rtl/` — 顶层与通用 Verilog

本目录放 **DE2-115 工程的核心 HDL**：顶层模块、时钟/复位/延时等 **与子目录无关的共用逻辑**。

## 建议在此目录编写的文件

| 文件（建议名） | 作用 |
|----------------|------|
| `top_de2_115.v` | 顶层：例化时钟生成、LCD、I2C、传感器调度、UI；将逻辑端口映射到 DE2-115 的 `CLOCK_50`、`KEY`、`GPIO` 等。 |
| `sync_reset.v`（可选） | 异步复位同步释放，避免亚稳态。 |
| `clk_enable_gen.v` | 由 50 MHz 派生 `spi_tick_en`、`i2c_tick_en`、`hz1_en` 等使能，供 SPI/I2C/秒时钟使用。 |
| `delay_cycles.v` 或 `delay_ns.v` | 参数化延时计数器；LCD 上电、RESET 脉冲、初始化间隔等会用到。 |

## 子目录

- [`lcd/`](lcd/) — ILI9488 显示与 SPI 物理层  
- [`i2c/`](i2c/) — I2C 主机与 BH1750 / DHT20  
- [`app/`](app/) — 传感器调度、时间、PIR、界面组合  

**约定**：具体外设驱动放在对应子目录；顶层只做 **连线、引脚、高层状态**（若需要）。
